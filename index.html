Overview

SDXL Studio is a complete, professional AI image generation application that combines the power of Stable Diffusion XL with a beautiful React interface. This documentation covers everything you need to set up, deploy, and use the system.

ℹ️ What You'll Build: A beautiful, professional image generation app running locally on your computer with a complete backend API in Google Colab.

System Architecture

Google Colab (Backend)
        ↓
Stable Diffusion XL (Model)
        ↓
ngrok Tunnel (Public API)
        ↓
React App (Frontend)
        ↓
Your Computer (Local)

Key Technologies

  • Frontend: React 18, Tailwind CSS, Lucide Icons
  • Backend: Stable Diffusion XL in Google Colab
  • API: FastAPI + ngrok tunnel
  • Deployment: Vercel, Netlify, or custom server
  • Version Control: Git & GitHub

Google Colab Setup

Google Colab provides free GPU access to run your AI model. This section covers the complete setup process.

Prerequisites

  • Google Account (Gmail)
  • HuggingFace account (for model access)
  • ngrok account (for public API)

Step-by-Step Setup

1

Create HuggingFace Account

Go to https://huggingface.co/settings/tokens and create a new read token for faster model downloads.

2

Create ngrok Account

Visit https://ngrok.com and create an account. Copy your authtoken from the dashboard.

3

Open Colab

Go to https://colab.research.google.com and upload your SDXL notebook.

4

Enable T4 GPU

Click Runtime → Change runtime type and select T4 GPU. This is critical!

5

Add Secrets

Click the 🔑 icon and add your HF_TOKEN and NGROK_AUTHTOKEN

6

Run Cells

Execute cells 0 → 1.1 → 1.2 → 1.3 → 3. Keep Cell 3 running!

⚠️ Important: Keep Cell 3 running while using your app. When you close Colab, your API stops working.

Expected Output

🚀 SDXL API is LIVE
=====================================================
Public URL   : https://abc-123-def.ngrok-free.app
Health check : https://abc-123-def.ngrok-free.app/health
Generate     : POST https://abc-123-def.ngrok-free.app/generate
=====================================================

Save this URL! You'll need it for your React app.

Local Development Setup

Run the SDXL Studio app locally on your computer using Node.js and React.

Installation Steps

1. Install Node.js

Download from https://nodejs.org (LTS version), install it, and restart your computer.

2. Create React App

npx create-react-app sdxl-studio
cd sdxl-studio

3. Add React Code

  • Open src/App.js
  • Delete all content
  • Paste the SDXL React code provided
  • Save the file

4. Install Dependencies

npm install lucide-react

5. Start Your App

npm start
✅ Success! Browser opens at http://localhost:3000

Using Your App

  1. Get your ngrok URL from Colab
  2. Paste into the "Connection" field
  3. Click "Test Connection"
  4. Enter a prompt and click "Generate"
  5. Wait 30-45 seconds for your image
  6. Download and enjoy! 🎉

React Application

The React app provides a beautiful interface for AI image generation with complete parameter control.

Parameter Guide

Parameter Default Range Effect
Steps 30 10-50 Higher = better quality, slower
Guidance Scale 7.5 1-15 Higher = stricter prompt adherence
Seed -1 Any -1=random, fixed=reproducible
Width/Height 1024 512, 1024 Image dimensions (optimized at 1024)

Prompt Engineering Tips

Use this structure for best results:

[Subject] + [Style] + [Lighting] + [Quality] + [Resolution]

Example:
"A majestic eagle soaring over snowy mountains, 
photorealistic, cinematic lighting, 
8k resolution, intricate details, masterfully crafted"

GitHub Setup & Version Control

Push your code to GitHub for backup, version control, and easy deployment.

Complete GitHub Workflow

1. Create GitHub Account

Go to https://github.com/signup and create your free account.

2. Install Git

Download from https://git-scm.com and install. Verify with git --version

3. Configure Git

git config --global user.name "Your Name"
git config --global user.email "your@email.com"

4. Initialize Project

cd sdxl-studio
git init
git add .
git commit -m "Initial commit: SDXL Studio"

5. Create Repository

Go to https://github.com/new and create a repo named sdxl-studio

6. Connect & Push

git remote add origin https://github.com/YOUR_USERNAME/sdxl-studio.git
git branch -M main
git push -u origin main

7. Verify & Share

Check your repo at https://github.com/YOUR_USERNAME/sdxl-studio

✅ Success! Your code is now backed up on GitHub and ready to share!

Frequently Asked Questions

General Questions

Q: Is this completely free?

A: Yes! Google Colab and ngrok free tiers provide everything you need. No credit card required.

Q: How long does image generation take?

A: 30-45 seconds per image on the free T4 GPU.

Q: Can I use images for commercial purposes?

A: Yes! The code is MIT licensed and images are yours to use.

Technical Questions

Q: What if my Colab session times out?

A: Restart it and run Cell 3 again. You'll get a new ngrok URL.

Q: Can I share my API with others?

A: Yes! Just give them your ngrok URL.

Troubleshooting

Q: "Cannot reach API" error

A: Make sure Colab is running, Cell 3 is active, and your URL is correct.

Q: "Port 3000 already in use"

A: Use npm start -- --port 3001

Still need help? Check the complete guides in your /outputs folder!
